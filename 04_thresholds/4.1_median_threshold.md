# Median Threshold

The median threshold splits patients into high- and low-risk groups at the median value of the predicted risk score.

## Definition

Given risk scores {r_1, ..., r_n}, the threshold is:

```
τ = median(r_1, ..., r_n)
```

Patients with r_i ≥ τ → high risk; r_i < τ → low risk.

## Properties

- Simple, reproducible, and assumption-free
- Always produces balanced groups (50/50 split)
- Does not require outcome data to set the threshold

## Limitations

- Balanced groups may not reflect the true clinical risk distribution
- The optimal cutpoint may not be at the median
- Can be misleading if the risk score distribution is skewed

## Implementation

```python
import numpy as np

tau = np.median(risk_scores_train)        # fit on TRAIN only
high_risk_test = risk_scores_test >= tau  # apply the frozen cutoff to TEST
```

Freezing the training median (rather than re-taking the test median) keeps the test split honest and the groups comparable across cohorts. Feed `high_risk_test` into the stratified KM plot in [`05_metrics/5.3_km_curves.md`](../05_metrics/5.3_km_curves.md).

## Use Cases

- Exploratory KM curve stratification
- Benchmarking other threshold methods
- When an external validation cohort is used (apply training median to test set)
