# pycox

`pycox` is a Python package for deep learning survival models built on top of PyTorch and `torchtuples`.

## Installation

```bash
pip install pycox
```

## Key Models

| Model | Loss | Notes |
|---|---|---|
| `CoxPH` (DeepSurv) | Cox partial likelihood | Classic deep Cox |
| `CoxCC` | Cox case-control | Memory-efficient approximation |
| `DeepHitSingle` | DeepHIT | Single event, discrete time |
| `DeepHit` | DeepHIT | Competing risks |
| `MTLR` | MTLR | Multi-task logistic regression |
| `PMF` | NLL | Piecewise constant hazard |

## Common Usage

```python
import pycox.models as models
import torchtuples as tt

net = tt.practical.MLPVanilla(in_features, [64, 64], out_features)
model = models.CoxPH(net, tt.optim.Adam)
model.fit(x_train, (durations_train, events_train), epochs=100)
surv = model.predict_surv_df(x_test)
```

## Best For

- Deep learning survival models
- Benchmarking multiple architectures
- Discrete-time and competing risk models
