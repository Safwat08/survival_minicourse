# lifelines

`lifelines` is the most popular Python package for classical survival analysis. It is pure Python, well-documented, and beginner-friendly.

## Installation

```bash
pip install lifelines
```

## Key Features

- Kaplan-Meier estimator
- Nelson-Aalen estimator
- Cox Proportional Hazards model (with L1/L2 regularization)
- Parametric models: Weibull, Log-Normal, Log-Logistic, etc.
- Log-rank test, statistical comparisons
- Plotting utilities

## Common Usage

```python
from lifelines import KaplanMeierFitter, CoxPHFitter
import pandas as pd

df = pd.DataFrame({"T": [...], "E": [...], "feature1": [...]})

# KM curve
kmf = KaplanMeierFitter()
kmf.fit(df["T"], df["E"])
kmf.plot_survival_function()

# Cox PH
cph = CoxPHFitter()
cph.fit(df, duration_col="T", event_col="E")
cph.print_summary()
```

## Best For

- Exploratory survival analysis
- Classical Cox PH on tabular data
- Statistical testing and visualization
- Clinical researchers new to survival analysis
