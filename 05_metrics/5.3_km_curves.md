# Kaplan-Meier Curves

The Kaplan-Meier (KM) estimator is the standard non-parametric method for estimating and visualizing the survival function.

## Estimator

```
S(t) = Π_{t_i ≤ t} (1 - d_i / n_i)
```

where d_i = number of events and n_i = number at risk at time t_i.

## In Model Evaluation

KM curves are used to:
1. **Visualize risk group separation**: plot KM for high vs. low risk groups defined by the model's threshold
2. **Assess calibration**: compare predicted S(t) to observed KM curve
3. **Communicate results**: preferred format for clinicians

## Log-Rank Test

When comparing two KM curves (e.g., high vs. low risk), use the log-rank test to assess statistical significance.

```python
from lifelines.statistics import logrank_test
result = logrank_test(T_high, T_low, E_high, E_low)
print(result.p_value)
```

## Plotting

```python
from lifelines import KaplanMeierFitter
kmf = KaplanMeierFitter()
kmf.fit(T, E, label="All patients")
kmf.plot_survival_function()
```

## Stratified Plot (the evaluation use case)

The clinically meaningful figure overlays the model's risk groups on one axis, with at-risk counts underneath (see [`references/6.2_lifelines.md`](../references/6.2_lifelines.md)):

```python
from lifelines import KaplanMeierFitter
from lifelines.plotting import add_at_risk_counts
import matplotlib.pyplot as plt

high = risk_scores >= tau                        # tau from 04_thresholds/
ax = plt.subplot()
km_hi = KaplanMeierFitter().fit(T[high],  E[high],  label="High risk")
km_lo = KaplanMeierFitter().fit(T[~high], E[~high], label="Low risk")
km_hi.plot_survival_function(ax=ax)
km_lo.plot_survival_function(ax=ax)
add_at_risk_counts(km_hi, km_lo, ax=ax)
```

Pair the figure with the [log-rank test](#log-rank-test) p-value, and choose `tau` via [`04_thresholds/`](../04_thresholds/4.1_median_threshold.md).

## Limitations

- Univariate — does not adjust for covariates
- Cannot extrapolate beyond the last observed event
