# scikit-survival

`scikit-survival` extends scikit-learn to support survival analysis, providing sklearn-compatible estimators for survival data.

## Installation

```bash
pip install scikit-survival
```

## Key Features

- Cox PH (with elastic net regularization)
- Survival SVM
- Gradient-boosted and component-wise boosted survival models
- Random Survival Forests
- Full sklearn pipeline compatibility (cross-validation, grid search, etc.)
- Evaluation metrics: C-index, Brier score, time-dependent AUC

## Common Usage

```python
from sksurv.linear_model import CoxPHSurvivalAnalysis
from sksurv.ensemble import RandomSurvivalForest
from sksurv.util import Surv

# Structured array format required
y_train = Surv.from_arrays(event=events_train.astype(bool),
                           time=times_train.astype(float))

model = CoxPHSurvivalAnalysis()
model.fit(X_train, y_train)
risk_scores = model.predict(X_test)
surv_functions = model.predict_survival_function(X_test)
```

## Best For

- Classical ML survival models (tree-based, regularized Cox)
- sklearn-compatible pipelines and cross-validation
- Censoring-aware evaluation metrics in sklearn-style workflows
- Radiomic feature survival models

For IPCW metrics, pass training outcomes to estimate the censoring distribution and
keep test follow-up/evaluation horizons within its supported range. Use pipelines so
imputation, encoding, scaling, feature selection, and model fitting are learned
inside each training fold rather than on the full dataset.
