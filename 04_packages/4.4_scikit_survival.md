# scikit-survival

`scikit-survival` extends scikit-learn to support survival analysis, providing sklearn-compatible estimators for survival data.

## Installation

```bash
pip install scikit-survival
```

## Key Features

- Cox PH (with elastic net regularization)
- Survival SVM
- Gradient boosted survival trees (GBM, GBSG)
- Random Survival Forests
- Full sklearn pipeline compatibility (cross-validation, grid search, etc.)
- Evaluation metrics: C-index, Brier score, time-dependent AUC

## Common Usage

```python
from sksurv.linear_model import CoxPHSurvivalAnalysis
from sksurv.ensemble import RandomSurvivalForest
import numpy as np

# Structured array format required
y = np.array([(bool(e), t) for e, t in zip(events, times)],
             dtype=[("event", bool), ("time", float)])

model = CoxPHSurvivalAnalysis()
model.fit(X_train, y_train)
risk_scores = model.predict(X_test)
```

## Best For

- Classical ML survival models (tree-based, regularized Cox)
- sklearn-compatible pipelines and cross-validation
- Evaluation metrics (best metrics library in the ecosystem)
- Radiomic feature survival models
