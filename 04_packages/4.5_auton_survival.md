# auton-survival

`auton-survival` (Carnegie Mellon / Auton Lab) is a toolkit for regression, counterfactual estimation, evaluation, and phenotyping with censored time-to-event data.

## Installation

```bash
git clone https://github.com/autonlab/auton-survival.git
pip install -r requirements.txt
```

## Key Features

- **Models**: DeepSurvivalMachines (DSM), DeepCoxPH, DeepCoxMixtures (DCM), DeepCoxMixturesHeterogeneousEffects (CMHE)
- **Phenotyping**: unsupervised, supervised, counterfactual, and virtual twins
- **Evaluation**: Brier score, IBS, AUC, time-dependent C-index — with bootstrapped confidence intervals
- **Treatment effects**: RMST, Risk at Time, Time at Risk between arms
- **Preprocessing**: imputation + scaling pipeline

## Common Usage

```python
from auton_survival import datasets, estimators

# Load data
outcomes, features = datasets.load_dataset("SUPPORT")

# Use the package's common estimator interface
model = estimators.SurvivalModel(model="dsm")
model.fit(features, outcomes)

# Predict risk at time horizons
predictions = model.predict_risk(features, times=[8, 12, 16])
```

## Cross-Validation Experiment

```python
from auton_survival.experiments import SurvivalRegressionCV

experiment = SurvivalRegressionCV(
    model="cph",
    num_folds=5,
    hyperparam_grid={"l2": [1e-4, 1e-3, 1e-2]},
)
model = experiment.fit(
    features, outcomes, horizons=[8, 12, 16], metric="ibs"
)
```

## Best For

- Deep survival models with competing risks (DSM)
- Counterfactual / causal survival analysis
- Treatment effect estimation
- Patient phenotyping and subgroup discovery

The counterfactual and treatment-effect utilities do not make an observational
analysis causal by themselves. Identification still requires a defensible treatment
assignment model, measured confounders, positivity, and appropriate sensitivity
analysis. The official installation path is currently source-based, so verify its
dependency compatibility in a clean environment before committing a project to it.
