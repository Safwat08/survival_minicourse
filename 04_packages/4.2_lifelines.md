# lifelines

`lifelines` provides classical survival estimators and regression models through a
DataFrame-oriented API. It is especially convenient for exploration, statistical
summaries, diagnostics, and publication-style plots.

## Installation

```bash
pip install lifelines
```

## Key Features

- Kaplan-Meier estimator
- Nelson-Aalen estimator
- Cox Proportional Hazards model (with L1/L2 regularization)
- Parametric models: Weibull, Log-Normal, Log-Logistic, etc.
- Aalen–Johansen cumulative incidence estimator for competing risks
- Log-rank test, statistical comparisons
- Proportional-hazards diagnostics and plotting utilities

## Common Usage

```python
from lifelines import KaplanMeierFitter, CoxPHFitter
import pandas as pd

df = pd.DataFrame({"T": [...], "E": [...], "feature1": [...]})

# KM curve
kmf = KaplanMeierFitter()
kmf.fit(df["T"], df["E"])
kmf.plot_survival_function()

# Cox PH
cph = CoxPHFitter()
cph.fit(df, duration_col="T", event_col="E")
cph.print_summary()
cph.check_assumptions(df, p_value_threshold=0.05)

# Predictions are different objects: relative risk versus absolute survival.
risk = cph.predict_partial_hazard(df)
surv = cph.predict_survival_function(df)
```

## Best For

- Exploratory survival analysis
- Classical Cox PH on tabular data
- Statistical testing and visualization
- Clinical researchers new to survival analysis

`lifelines` is not a neural-network framework and its built-in concordance measure
does not replace the IPCW evaluation suite in
[`scikit-survival`](4.4_scikit_survival.md). For competing risks, use
`AalenJohansenFitter` for descriptive CIFs rather than $1-\hat S(t)$.
