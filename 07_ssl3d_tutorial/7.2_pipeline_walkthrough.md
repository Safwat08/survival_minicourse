# SSL3D-Survival: Pipeline Walkthrough

Step-by-step guide to running the SSL3D-Survival pipeline.

## Step 1: Data Preparation

```python
# Expected format
dataset = [
    {"image": "path/to/volume.nii.gz", "time": 24.5, "event": 1},
    {"image": "path/to/volume.nii.gz", "time": 36.0, "event": 0},  # censored
    ...
]
```

Preprocessing:
- Resampling to isotropic spacing (e.g., 1mm³)
- Intensity normalization (z-score or min-max)
- Cropping / padding to fixed size (e.g., 128³)

## Step 2: Load Pretrained Encoder

```python
from ssl3d import load_pretrained_encoder

encoder = load_pretrained_encoder("ssl3d_imagenet_pretrained")
encoder.eval()
```

## Step 3: Extract Features

```python
import torch

features = []
for sample in dataloader:
    with torch.no_grad():
        feat = encoder(sample["image"].cuda())  # (B, D)
    features.append(feat.cpu())

features = torch.cat(features)  # (N, D)
```

## Step 4: Fine-tune with Survival Loss

```python
from torchsurv.loss import cox
import torch.nn as nn

survival_head = nn.Linear(encoder.embed_dim, 1)
optimizer = torch.optim.Adam(
    list(encoder.parameters()) + list(survival_head.parameters()), lr=1e-4
)

for epoch in range(100):
    log_hz = survival_head(encoder(images)).squeeze()
    loss = cox.neg_partial_log_likelihood(log_hz, events, times)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

## Step 6: Stratify and Plot KM Curves

```python
from lifelines import KaplanMeierFitter
from lifelines.statistics import logrank_test

median_risk = risk_scores.median()
high = risk_scores >= median_risk

kmf = KaplanMeierFitter()
ax = kmf.fit(times[high], events[high], label="High risk").plot()
kmf.fit(times[~high], events[~high], label="Low risk").plot(ax=ax)

result = logrank_test(times[high], times[~high], events[high], events[~high])
print(f"Log-rank p = {result.p_value:.4f}")
```
