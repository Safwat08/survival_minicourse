# SSL3D-Survival: Overview

SSL3D-Survival extends self-supervised learning (SSL) pretraining on 3D medical images to survival analysis tasks.

## Motivation

3D medical imaging datasets paired with survival outcomes are small. Self-supervised pretraining on large unlabeled image collections learns rich volumetric representations that can be fine-tuned for survival prediction with limited labeled data.

## Pipeline

```
3D Images (unlabeled)
        │
        ▼
SSL Pretraining (e.g., contrastive, masked autoencoder)
        │
        ▼
Pretrained 3D Encoder
        │
        ▼
Fine-tuning with Survival Loss (Cox / DeepHIT)
  + (T, δ) labels
        │
        ▼
Risk Score per Patient
        │
        ▼
Evaluation: C-index, Brier Score, KM curves
```

## Key Components

- **3D Encoder**: 3D CNN or 3D ViT backbone
- **SSL Pretraining**: SimCLR, MoCo, DINO, or MAE adapted for 3D volumes
- **Survival Head**: linear layer or MLP mapping features → risk score
- **Loss**: Cox PH partial likelihood or DeepHIT loss
