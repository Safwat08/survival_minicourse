# Negative Log-Likelihood Loss

The NLL loss is used in fully parametric survival models, where the survival time distribution is explicitly specified.

## Common Distributions

| Distribution | Use case |
|---|---|
| Exponential | Constant hazard |
| Weibull | Monotone increasing/decreasing hazard |
| Log-normal | Non-monotone hazard |
| Log-logistic | Non-monotone, closed-form survival |

## Loss Function

For subject *i* with observed time T_i and event indicator δ_i:

```
ℓ_i = δ_i * log f(T_i | x_i) + (1 - δ_i) * log S(T_i | x_i)
```

- Event observed (δ=1): use the PDF f(t)
- Censored (δ=0): use the survival function S(t)

Total loss: L = -Σ ℓ_i

## In Deep Learning

Neural networks parameterize the distribution (e.g., predict Weibull shape and scale). Used in **DeepHazard**, **Nnet-survival**, and parametric variants of survival networks.

## Implementation

There are two NLL flavors: **fully parametric** (predict distribution parameters) and **discrete-time** (predict a hazard/PMF over time bins).

**torchsurv** — parametric Weibull AFT loss for a custom model (see [`references/6.3_torchsurv.md`](../references/6.3_torchsurv.md)):

```python
from torchsurv.loss import weibull

log_params = model(x)             # network predicts log shape & log scale
loss = weibull.neg_log_likelihood(log_params, events, times)
```

**pycox** — discrete-time NLL models; the continuous time axis is binned first via a label transform (see [`references/6.1_pycox.md`](../references/6.1_pycox.md)):

```python
import pycox.models as models
# LogisticHazard / PMF / MTLR all minimise a discrete-time NLL
labtrans = models.LogisticHazard.label_transform(num_durations=20)
y_train = labtrans.fit_transform(durations, events)
model = models.LogisticHazard(net, optimizer, duration_index=labtrans.cuts)
```

**lifelines** — classical parametric fitters, one per distribution (see [`references/6.2_lifelines.md`](../references/6.2_lifelines.md)):

```python
from lifelines import WeibullFitter, LogNormalFitter, WeibullAFTFitter
WeibullFitter().fit(T, E)                          # univariate
WeibullAFTFitter().fit(df, "T", "E")               # with covariates
```

> **Parametric vs. Cox.** Parametric NLL gives you a full predicted S(t) and can *extrapolate* beyond the last event — at the cost of a distributional assumption Cox avoids. Use it when you need absolute event probabilities, not just a ranking.
